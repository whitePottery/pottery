# Установка

Запустить команду в терминале из папки проекта

```
composer require whitePottery/pottery
```

## 1. Создание пакета

Создать свой пакет

 ```
   php artisan pottery:make
 ```
следуем инструкциям

после выполнения пакет с приставкой "_TMP" появится в папке vendor
